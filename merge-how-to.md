Получаем изменение с сервера с помощью команды:

`git fetch`


</br>
Переключимся на ветку друга:

`git checkout surname`

</br>
Переключимся на основную ветку:

`git checkout master`

</br>
Обьединить ветку друга с основной веткой:

`git merge surname`

</br>
Отправим изменения на сервер:

`git push`
